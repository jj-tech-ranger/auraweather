A fast, modern, and installable Progressive Web App (PWA) for real-time weather monitoring, built with Next.js 15 and Tailwind CSS.

## ‚ú® Features

- **Real-Time Weather Dashboard:** Get instant temperature, humidity, wind speed, and atmospheric pressure based on your live GPS location.
- **7-Day Interactive Forecast:** A detailed weekly view predicting temperature highs/lows, precipitation chances, and UV indices.
- **Dynamic Weather Map:** Visual overlays for global cloud cover, precipitation, temperature distribution, and wind patterns using live satellite data.
- **Progressive Web App (PWA):** Fully installable on iOS, Android, and Desktop environments for a native app-like experience.
- **Dark & Light Mode:** Seamlessly switch between a bright, clean interface and a deep, immersive dark mode.
- **Global Search:** Check the weather in any city worldwide with instant API fetching.
- **Smart Fallback:** Intelligently defaults to Nairobi, Kenya if GPS access is unavailable or denied.

## üöÄ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router, Client Components)
- **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/) & CSS Modules

- **Icons:** [Lucide React](https://lucide.dev/)
- **Data Source:** [OpenWeatherMap API](https://openweathermap.org/) (Weather, Forecast, Air Quality, UV, and Map Tiles)
- **PWA Integration:** `@ducanh2912/next-pwa`
- **Deployment:** [Vercel](https://vercel.com/)
